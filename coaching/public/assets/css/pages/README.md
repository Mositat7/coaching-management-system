# استایل‌های مخصوص هر صفحه (Pages CSS)

هر فایل این پوشه مربوط به **یک صفحه یا بخش** است و فقط در همان ویو لود می‌شود.

## قرار دادن در ویو

در فایل blade مربوطه، داخل `@section('head')` فقط یک لینک به فایل CSS بگذارید:

```blade
@section('head')
    <link href="{{ asset('assets/css/pages/نام-فایل.css') }}" rel="stylesheet">
@endsection
```

## قرارداد نام‌گذاری

- یک فایل به ازای هر صفحه/بخش، مثلاً: `workout-categories.css`، `coach-add.css`
- نام فایل با مسیر ویو هماهنگ باشد تا پیدا کردن راحت باشد

## مزایا

- صفحهٔ blade تمیز و خواناتر
- CSS قابل کش توسط مرورگر
- نگهداری و تغییر استایل در یک جا
- بدون وابستگی به اینترنت (همه از `asset()` لوکال)
